Bokuchi is available for Windows, macOS, and Linux. Download the latest version from the [GitHub Releases page](https://github.com/Bokuchi-Editor/bokuchi/releases/latest).

## Download

Visit the [Releases page](https://github.com/Bokuchi-Editor/bokuchi/releases/latest) and download the installer for your operating system.

<Tabs>
  <TabItem label="Windows">
    - **`.exe`** — Standard installer (recommended)
    - **`.msi`** — MSI installer

    Run the downloaded installer and follow the on-screen instructions.
  </TabItem>
  <TabItem label="macOS">
    - **`.dmg`** — Disk image (Universal, works on both Intel and Apple Silicon)

    1. Open the `.dmg` file
    2. Drag **Bokuchi** to the **Applications** folder
    3. Open Bokuchi from your Applications folder

    <img src={imgMacDmg.src} alt="macOS DMG installer — drag Bokuchi to Applications" width="600" />
  </TabItem>
  <TabItem label="Linux">
    - **`.deb`** — Debian/Ubuntu package
    - **`.AppImage`** — Portable, runs on most distributions

    For `.deb`:
    ```bash
    sudo dpkg -i bokuchi_*.deb
    ```

    For `.AppImage`:
    ```bash
    chmod +x Bokuchi_*.AppImage
    ./Bokuchi_*.AppImage
    ```
  </TabItem>
</Tabs>

## Windows: SmartScreen Warning

When you download and run the Bokuchi installer on Windows, you may see a series of warnings from the browser and Microsoft Defender SmartScreen.

<Steps>
1. **Browser download warning** — Your browser may show a warning that the file is not commonly downloaded. Click the menu or **More** option to proceed.

   <img src={imgSmartscreen01.src} alt="Browser download warning" width="600" />

2. **Save the file** — Select **Save** (or **Keep**) from the menu to save the installer to your computer.

   <img src={imgSmartscreen02.src} alt="Select Save to keep the file" width="600" />

3. **SmartScreen confirmation** — Microsoft Defender SmartScreen may show a final confirmation dialog. Click **Keep** to allow the file.

   <img src={imgSmartscreen03.src} alt="SmartScreen confirmation — click Keep" width="600" />
</Steps>

<Aside>
This warning appears because Bokuchi is not signed with a commercial code-signing certificate. The app is open source and safe to use. The warning will only appear on the first installation.
</Aside>

## After Installation

When you launch Bokuchi for the first time, you will see the editor with an empty document ready for you to start writing.

<img src={imgFirstLaunch.src} alt="Bokuchi first launch screen" width="700" />

The interface language will match your system language if it is one of the [14 supported languages](/reference/supported-languages/). You can change it at any time in [Settings](/guides/settings/).
