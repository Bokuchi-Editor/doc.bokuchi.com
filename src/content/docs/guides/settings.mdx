Open the settings panel with **Cmd+,** (macOS) or **Ctrl+,** (Windows/Linux). You can also press **F1** and access settings from the help menu.

## Settings Categories

### Editor

| Setting | Description | Default |
|---------|-------------|---------|
| Font size | Editor text size | 14 |
| Line numbers | Show line numbers in the gutter | On |
| Tab size | Number of spaces per tab | 2 |
| Word wrap | Wrap long lines to fit the editor width | On |
| Minimap | Show a minimap overview of the document | On |
| Formatting bar | Show the [Markdown formatting toolbar](/guides/markdown-toolbar/) above the editor | On |

<img src={imgEditor.src} alt="Settings panel — Editor tab" width="700" />

### Appearance

| Setting | Description | Default |
|---------|-------------|---------|
| Theme | Choose from 9 built-in themes | Default |

See [Themes](/guides/themes/) for details on each theme.

### Interface

| Setting | Description | Default |
|---------|-------------|---------|
| Language | UI language (14 languages available) | English |
| Tab layout | Horizontal or vertical tabs | Horizontal |
| Tab close button position | Show close/pin icon on left or right of tab | Right |
| New tab button position | Where the new tab (+) button sits in vertical tabs: top or bottom | Top |
| Zoom level | Interface zoom | 100% |
| Scroll sync | Editor-to-preview, bidirectional, or off | Bidirectional |
| Outline display mode | Persistent or overlay | Persistent |
| Explorer display mode | Persistent, overlay, or off | Off |
| File filter | Show Markdown files only or all files | Markdown only |

<img src={imgInterface.src} alt="Settings panel — Interface tab" width="700" />

### Advanced

| Setting | Description | Default |
|---------|-------------|---------|
| Auto save | Save files automatically after 3 seconds of inactivity | On |
| Show whitespace | Display invisible characters | Off |
| Table conversion | How to handle pasted table data: auto, confirm, or off | Confirm |
| Table layout (Preview) | How table columns are sized in the preview and exported HTML: equal widths, fit-to-content with wrapping, or fit-to-content with horizontal scroll | Fit to content (wrap) |

#### Rendering Extensions

| Setting | Description | Default |
|---------|-------------|---------|
| KaTeX (Math Expressions) | Render math using `$...$` (inline) and `$$...$$` (display) syntax | On |
| Mermaid (Diagrams) | Render diagrams from ` ```mermaid ` fenced code blocks | Off |
| Marp (Presentations) | Render Markdown with `marp: true` front-matter as slide presentations | Off |

Disabled features do not load their libraries, keeping the editor lightweight.

When Marp is enabled, a **Marp Theme Folder** option also appears here, letting you register custom slide themes from a folder of CSS files. See [Custom Themes](/guides/marp/#custom-themes).

<Aside>
If you open a file that uses KaTeX, Mermaid, or Marp content while the matching option above is off, Bokuchi shows a one-time notice pointing to the relevant setting — so it's easy to spot when math, diagrams, or slides aren't rendering simply because the feature is disabled.
</Aside>

### Global Variables

Manage your [global variables](/guides/variables/) — add, edit, delete, import, and export.

## Export and Import Settings

You can export all your settings as a JSON file and import them on another machine. This is useful for backing up your configuration or sharing it.

- **Export:** Click the Export button in the settings panel
- **Import:** Click the Import button and select a previously exported JSON file

## Reset Settings

If you want to start fresh, use the Reset button to restore all settings to their defaults.

<Aside>
Resetting settings will not delete your global variables. Variable management is handled separately.
</Aside>
