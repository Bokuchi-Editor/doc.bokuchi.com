## Supported Operating Systems

| OS | Supported Versions |
|----|--------------------|
| **Windows** | Windows 10 and later |
| **macOS** | macOS 10.15 (Catalina) and later (Intel and Apple Silicon) |
| **Linux** | Ubuntu 20.04+, Debian 11+, and other modern distributions |

## Hardware Requirements

Bokuchi is a lightweight application. It runs well on virtually any modern computer.

- **Disk space:** Approximately 100 MB
- **Memory:** Minimal — Bokuchi uses significantly less RAM than Electron-based editors

## Key Characteristics

- **Completely offline** — No internet connection is required. Bokuchi does not send any data over the network.
- **Single instance** — Only one instance of Bokuchi can run at a time. If you try to open a second instance, the existing window will be brought to the foreground.
- **Native performance** — Built with Tauri and Rust, Bokuchi uses the system's native web view for rendering, resulting in a small footprint and fast startup.
