description: Complete list of keyboard shortcuts available in Bokuchi.

## File Operations

| Action | macOS | Windows / Linux |
|--------|-------|-----------------|
| New file | Cmd+N | Ctrl+N |
| Open file | Cmd+O | Ctrl+O |
| Save | Cmd+S | Ctrl+S |
| Save as | Cmd+Shift+S | Ctrl+Shift+S |
| Recent files | Cmd+R | Ctrl+R |

## View

| Action | macOS | Windows / Linux |
|--------|-------|-----------------|
| Split view | Ctrl+Shift+1 | Ctrl+Shift+1 |
| Editor only | Ctrl+Shift+2 | Ctrl+Shift+2 |
| Preview only | Ctrl+Shift+3 | Ctrl+Shift+3 |
| Cycle view modes | Ctrl+Shift+V | Ctrl+Shift+V |
| Zoom in | Cmd+Plus (=) | Ctrl+Plus (=) |
| Zoom out | Cmd+Minus | Ctrl+Minus |
| Reset zoom | Cmd+0 | Ctrl+0 |

## Panels

| Action | macOS | Windows / Linux |
|--------|-------|-----------------|
| Toggle explorer | Ctrl+Shift+E | Ctrl+Shift+E |
| Toggle outline | Ctrl+Shift+O | Ctrl+Shift+O |

## Navigation

| Action | macOS | Windows / Linux |
|--------|-------|-----------------|
| Next tab | Ctrl+Tab | Ctrl+Tab |
| Previous tab | Ctrl+Shift+Tab | Ctrl+Shift+Tab |

## Other

| Action | macOS | Windows / Linux |
|--------|-------|-----------------|
| Settings | Cmd+, | Ctrl+, |
| Help | F1 | F1 |
