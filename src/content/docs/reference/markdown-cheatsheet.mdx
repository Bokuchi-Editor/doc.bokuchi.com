A printable quick reference for the most common Markdown syntax.

## Text Formatting

| Syntax | Result |
|--------|--------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strikethrough~~` | ~~strikethrough~~ |
| `` `inline code` `` | `inline code` |

## Headings

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

## Lists

```markdown
- Bullet item
- Another item
  - Nested item

1. Numbered item
2. Another item

- [x] Checked task
- [ ] Unchecked task
```

## Links and Images

```markdown
[Link text](https://example.com)
![Image alt text](image.png)
```

## Blockquotes

```markdown
> This is a quote.
> It can span multiple lines.
```

## Code Blocks

````markdown
```language
code here
```
````

## Tables

```markdown
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```

## Horizontal Rule

```markdown
---
```

## Bokuchi-Specific: Variables

```markdown
<!-- @var name: value -->
Text with {{name}} placeholder.
```

See [Variables](/guides/variables/) for full documentation.
