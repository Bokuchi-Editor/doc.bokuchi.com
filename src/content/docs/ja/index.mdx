## Bokuchi の特長

<CardGrid stagger>
  <Card title="完全オフライン" icon="laptop">
    インターネット接続は不要です。ファイルはすべてお使いのパソコンに保存されます。クラウド連携やアカウント登録はありません。
  </Card>
  <Card title="リアルタイムプレビュー" icon="document">
    入力した Markdown がすぐに整形されて表示されます。分割表示・エディタのみ・プレビューのみの3つのモードから選べます。
  </Card>
  <Card title="マルチタブ編集" icon="list-format">
    複数のファイルをタブで同時に開いて管理できます。ドラッグ＆ドロップで並べ替え、セッションは自動復元されます。
  </Card>
  <Card title="豊富な機能" icon="setting">
    変数システム、テーブル変換、フォルダツリー、アウトラインパネル、5種類のテーマ、14言語のUI対応など、すべて内蔵しています。
  </Card>
</CardGrid>
