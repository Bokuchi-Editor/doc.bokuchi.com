## Bokuchi の特長

<CardGrid stagger>
  <Card title="完全オフライン" icon="laptop">
    インターネット接続しなくても使用可能です。ファイルはすべてお使いのパソコンに保存されます。作者がオフラインでも使いたいこと、またセキュリティ上の理由からクラウド連携やアカウント登録はありません。今後の実装予定も今のところ考えておりません。
  </Card>
  <Card title="リアルタイムプレビュー" icon="document">
    リアルタイムで、入力した Markdown
    はすぐに整形表示されます。分割表示・エディタのみ・プレビューのみの3つのビューモードから選べます。
  </Card>
  <Card title="マルチタブ編集" icon="list-format">
    複数のファイルをタブで同時に開いて管理できます。ドラッグ＆ドロップで並べ替え可能。アプリを終了しても、次回の起動時にセッションは自動復元されます。
  </Card>
  <Card title="豊富な機能" icon="setting">
    変数システム、テーブル変換、フォルダツリー、アウトラインパネル、KaTeX 数式、Mermaid 図表、5種類のテーマ、14言語のUI対応など。
  </Card>
</CardGrid>
