## 一般

### Markdown とは何ですか？

Markdown は、シンプルな記号を使ってテキストを装飾する書き方です。たとえば `**太字**` で**太字**に、`# 見出し` で見出しになります。Bokuchi では記法を覚えなくても、[ツールバー](/ja/guides/markdown-toolbar/)のボタンで装飾できます。

### Bokuchi は無料ですか？

はい。Bokuchi は無料のオープンソースソフトウェアです。

### インターネット接続は必要ですか？

不要です。Bokuchi は完全にオフラインで動作します。ファイルはお使いのパソコンにローカル保存され、ネットワークを通じてデータが送信されることはありません。

## ファイル

### ファイルはどこに保存されますか？

ユーザーが指定した場所に保存されます。Bokuchi は独自の保存場所やクラウドサービスを使用しません。

### 他のエディタで作った Markdown ファイルは開けますか？

はい。Bokuchi は `.md`、`.txt`、`.markdown` ファイルに対応しています。標準的な Markdown ファイルであれば問題なく開けます。

### 自動保存はありますか？

はい。オートセーブはデフォルトで有効になっており、入力を停止してから 3 秒後に保存されます。[設定](/ja/guides/settings/) > 詳細設定 で ON/OFF を切り替えできます。

### アプリを閉じてもタブは復元されますか？

はい。開いていたすべてのタブ、その内容、設定は Bokuchi を再度開いた際に自動的に復元されます。

## 外観

### テーマを変更するには？

[設定](/ja/guides/settings/) > 外観 で変更するか、ウィンドウ下部のステータスバーに表示されているテーマ名をクリックしてください。詳しくは[テーマの変更](/ja/guides/themes/)をご覧ください。

### 言語を変更するには？

[設定](/ja/guides/settings/) > インターフェース > 言語 で変更できます。Bokuchi は [14 言語](/ja/reference/supported-languages/)に対応しています。

## トラブルシューティング

### Windows で SmartScreen に「発行元不明」と表示されます

商用のコード署名証明書で署名されていないアプリでは通常の挙動です。**詳細情報** → **実行** の順にクリックしてください。詳しい手順は[インストールガイド](/ja/getting-started/installation/)をご覧ください。

### PDF 出力はできますか？

現在は [HTML エクスポート](/ja/guides/export/)に対応しています。PDF 出力は将来のバージョンで実装予定です。

## 機能

### 変数とは何ですか？

変数を使うと、値（著者名や日付など）を一箇所で定義して、ドキュメント内の複数箇所で `{{変数名}}` として使い回せます。詳しくは[変数システム](/ja/guides/variables/)をご覧ください。

### Excel や Web ページからテーブルを貼り付けられますか？

はい。Bokuchi は HTML テーブルや CSV/TSV データをペースト時に自動的に Markdown テーブルに変換できます。詳しくは[テーブル変換](/ja/guides/table-conversion/)をご覧ください。
