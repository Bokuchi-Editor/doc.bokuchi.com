このガイドでは、Bokuchi を開いてから最初のファイルを保存するまでの一連の流れをご紹介します。

## Markdown とは？

Markdown（マークダウン）は、特殊な記号を使ってテキストを装飾する書き方です。たとえば `# こんにちは` と書くと大きな見出しになり、`**太字**` と書くと**太字**になります。

Bokuchi では、左側（エディタ）に Markdown を入力すると、右側（プレビュー）に整形された結果がリアルタイムで表示されます。

<Aside>
Markdown の記法を覚える必要はありません。Bokuchi には[ツールバー](/ja/guides/markdown-toolbar/)があり、ボタンを押すだけで装飾できます。
</Aside>

## 最初のドキュメントを作成する

<Steps>
1. **Bokuchi を起動する**

   アプリケーションフォルダ（macOS）、スタートメニュー（Windows）、またはアプリランチャー（Linux）から Bokuchi を開きます。空の新規ドキュメントが用意されています。

2. **テキストを入力する**

   左側のパネル（エディタ）をクリックして、テキストを入力してみましょう：
   ```markdown
   # はじめてのドキュメント

   こんにちは！Bokuchi で書いた最初のドキュメントです。

   - 項目1
   - 項目2
   - **太字の項目**
   ```

   <img src={imgTyping.src} alt="エディタに Markdown を入力しているところ" width="700" />

3. **プレビューを確認する**

   右側のパネルを見てください。入力したテキストが装飾されて表示されています。見出しは大きく、リストには箇条書きマークが付き、太字が目立っています。

   <img src={imgPreview.src} alt="エディタとプレビューの分割表示" width="700" />

4. **ドキュメントを保存する**

   **Cmd+S**（macOS）または **Ctrl+S**（Windows/Linux）を押します。保存ダイアログが表示されるので、保存場所を選び、ファイル名（例：`はじめてのメモ.md`）を入力して**保存**をクリックします。

   <img src={imgSave.src} alt="保存ダイアログ" width="700" />
</Steps>

## 何が起こったか

- **エディタ**（左パネル）に Markdown の書式でテキストを入力しました
- Bokuchi が**プレビュー**（右パネル）にリアルタイムで整形結果を表示しました
- ファイルを `.md`（Markdown）ファイルとしてパソコンに保存しました

これが基本の流れです。作成したファイルはすべてお使いのパソコンに保存されます。どこにもアップロードされることはありません。
