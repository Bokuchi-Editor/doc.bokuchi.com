Bokuchi は Windows、macOS、Linux に対応しています。最新版は [GitHub Releases ページ](https://github.com/Bokuchi-Editor/bokuchi/releases/latest)からダウンロードできます。

## ダウンロード

[Releases ページ](https://github.com/Bokuchi-Editor/bokuchi/releases/latest)にアクセスし、お使いの OS に合ったインストーラーをダウンロードしてください。

<Tabs>
  <TabItem label="Windows">
    - **`.exe`** — 標準インストーラー（推奨）
    - **`.msi`** — MSI インストーラー

    ダウンロードしたインストーラーを実行し、画面の指示に従ってください。
  </TabItem>
  <TabItem label="macOS">
    - **`.dmg`** — ディスクイメージ（ユニバーサル版。Intel / Apple Silicon 両対応）

    1. `.dmg` ファイルを開きます
    2. **Bokuchi** を **アプリケーション** フォルダにドラッグします
    3. アプリケーションフォルダから Bokuchi を起動します
  </TabItem>
  <TabItem label="Linux">
    - **`.deb`** — Debian / Ubuntu パッケージ
    - **`.AppImage`** — ポータブル版（多くのディストリビューションで動作）

    `.deb` の場合：
    ```bash
    sudo dpkg -i bokuchi_*.deb
    ```

    `.AppImage` の場合：
    ```bash
    chmod +x Bokuchi_*.AppImage
    ./Bokuchi_*.AppImage
    ```
  </TabItem>
</Tabs>

## Windows：SmartScreen の警告について

Windows で Bokuchi のインストーラーを実行すると、**Microsoft Defender SmartScreen** の警告が表示され、「Windows によって PC が保護されました」や「発行元不明」と表示されることがあります。

<Steps>
1. SmartScreen のダイアログで **詳細情報** をクリックします
2. 表示された **実行** ボタンをクリックします
3. インストーラーが正常に進行します
</Steps>

<Aside>
この警告は、Bokuchi が商用のコード署名証明書で署名されていないために表示されます。Bokuchi はオープンソースソフトウェアであり、安全にお使いいただけます。初回インストール時のみ表示されます。
</Aside>

## インストール後

Bokuchi を初めて起動すると、空のドキュメントが開いた状態のエディタが表示されます。すぐに文章を書き始めることができます。

インターフェースの言語は、お使いのシステム言語が[対応14言語](/ja/reference/supported-languages/)に含まれていれば自動的に設定されます。[設定](/ja/guides/settings/)からいつでも変更できます。
