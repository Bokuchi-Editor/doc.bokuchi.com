## ファイル操作

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 新規作成 | Cmd+N | Ctrl+N |
| 開く | Cmd+O | Ctrl+O |
| 保存 | Cmd+S | Ctrl+S |
| 名前を付けて保存 | Cmd+Shift+S | Ctrl+Shift+S |
| 最近使ったファイル | Cmd+R | Ctrl+R |

## 表示

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 分割ビュー | Ctrl+Shift+1 | Ctrl+Shift+1 |
| エディタのみ | Ctrl+Shift+2 | Ctrl+Shift+2 |
| プレビューのみ | Ctrl+Shift+3 | Ctrl+Shift+3 |
| 表示モード切替 | Cmd+Shift+D | Ctrl+Shift+D |
| ズームイン | Cmd+Plus (=) | Ctrl+Plus (=) |
| ズームアウト | Cmd+Minus | Ctrl+Minus |
| ズームリセット | Cmd+0 | Ctrl+0 |

## パネル

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| エクスプローラー | Ctrl+Shift+E | Ctrl+Shift+E |
| アウトライン | Ctrl+Shift+O | Ctrl+Shift+O |

## ナビゲーション

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 次のタブ | Ctrl+Tab | Ctrl+Tab |
| 前のタブ | Ctrl+Shift+Tab | Ctrl+Shift+Tab |

## 編集

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| カーソル位置のテーブルを整形／編集 | Cmd+Shift+L | Ctrl+Shift+L |

## その他

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 設定 | Cmd+, | Ctrl+, |
| ヘルプ | F1 | F1 |
| 集中モード（臨）を抜ける | Esc | Esc |
