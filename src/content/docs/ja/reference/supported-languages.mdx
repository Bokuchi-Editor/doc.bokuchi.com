Bokuchi のユーザーインターフェースは 14 言語に対応しています。システム言語を自動検出しますが、[設定](/ja/guides/settings/) > インターフェース > 言語 で手動変更もできます。

## 言語の変更方法

1. **設定** を開きます（macOS: Cmd+, / Windows/Linux: Ctrl+,）
2. **インターフェース** セクションに移動します
3. ドロップダウンからお好みの言語を選択します
4. インターフェースが即座に更新されます（再起動は不要）
