## 対応 OS

| OS | 対応バージョン |
|----|----------------|
| **Windows** | Windows 10 以降 |
| **macOS** | macOS 10.15 (Catalina) 以降（Intel / Apple Silicon 対応） |
| **Linux** | Ubuntu 20.04+、Debian 11+ 等のモダンなディストリビューション |

## ハードウェア要件

Bokuchi は軽量なアプリケーションです。一般的な現代のパソコンであれば問題なく動作します。

- **ディスク容量：** 約 100 MB
- **メモリ：** 最小限 — Electron ベースのエディタと比較して、大幅に少ないメモリ使用量

## 主な特徴

- **完全オフライン** — インターネット接続は不要です。Bokuchi はネットワーク経由でデータを送信しません。
- **シングルインスタンス** — Bokuchi は同時に1つだけ起動できます。2つ目を起動しようとすると、既存のウィンドウが前面に表示されます。
- **ネイティブパフォーマンス** — Tauri と Rust で構築されており、システムのネイティブ Web ビューを使用するため、軽量かつ高速に起動します。
