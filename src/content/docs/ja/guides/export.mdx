Bokuchi は Markdown ドキュメントをスタイル付き HTML ファイルとしてエクスポートできます。Web ブラウザで開いたり、他の人と共有したりするのに便利です。

## HTML エクスポート

エクスポートされた HTML ファイルには以下が含まれます：

- **スタイル付き** — Bokuchi のプレビューと同じ見た目で出力されます
- **シンタックスハイライト** — コードブロックのカラーハイライトが保持されます
- **画像の埋め込み** — 相対パスで参照されたローカル画像はデータ URI として埋め込まれるため、HTML ファイル単体で完結します
- **変数の展開** — すべての[変数](/ja/guides/variables/)が実際の値に置き換えられます
- **数式と図表** — KaTeX の数式と Mermaid の図表がレンダリングされた状態で出力されます

## エクスポートの手順

1. エクスポートしたいファイルを開きます
2. メニューからエクスポートを選択します
3. 保存場所とファイル名を選びます
4. `.html` ファイルが保存されます

<Aside>
エクスポートされた HTML ファイルは自己完結型です。受け取った人は Web ブラウザで開くだけで、Bokuchi やその他のソフトウェアは必要ありません。
</Aside>

## その他のエクスポート

### 設定のエクスポート / インポート

Bokuchi の設定を JSON ファイルとしてエクスポートしたり、ファイルからインポートしたりできます。設定の複数マシン間での同期やバックアップに便利です。

### 変数のエクスポート / インポート

グローバル変数を YAML ファイルとしてエクスポート、またはインポートできます。チームでの変数共有に便利です。
