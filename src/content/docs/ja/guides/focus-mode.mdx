**臨（Rin）** は、Bokuchi の集中のためのモードです。ヘッダー、整形ツールバー、タブのサイドバー、サイドパネルなど、エディタの周囲をすべて隠し、文章だけが残ります。

<img src="/images/guides/focus-mode-rin-01.gif" alt="集中モード「臨」に入る様子。周囲の UI がスライドして消え、全幅エディタと右上の Exit Rin ボタンが残る" width="700" />

## 集中モードに入る

上部ツールバーの **臨** ボタン（臨の書道風アイコン）をクリックします。

エディタの周囲がスライドして消え、編集領域がウィンドウいっぱいに広がります。

<img src={imgButton.src} alt="アプリヘッダーの臨ボタンのクローズアップ" width="700" />

## 集中モードを抜ける

抜ける方法は 2 つあります。

- **Esc** を押す
- 右上の **Exit Rin** ボタンをクリックする

終了ボタンは、入力を始めるとすぐに薄くなって邪魔にならず、マウスを動かすと再び表示されます。

<Aside>
集中モードを抜けると、直前の表示モード（分割／エディタ／プレビュー）と、開いていたパネルが元に戻ります。
</Aside>

## 集中モード中の操作

- 終了ボタンのすぐ下にある **エディタ幅の切り替え** ボタンで、エディタを全幅と、やや狭い中央寄せの 1 段組みとで切り替えられます。長い文章を書くときに、読みやすい行幅にするのに便利です。
- 整形ツールバーは、通常は表示する設定でも自動的に隠れます。モードを抜けると[設定](/ja/guides/settings/)の値に戻ります。

集中モードはセッションごとの一時的な状態で、再起動後には保持されません。
