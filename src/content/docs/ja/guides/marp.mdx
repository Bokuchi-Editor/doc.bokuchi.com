Bokuchi は [Marp](https://marp.app/) を使い、Markdown ファイルをスライドプレゼンテーションとしてレンダリングできます。Markdown でスライドを書き、プレゼンテーションとしてプレビューできます。別途プレゼンテーションソフトは不要です。

<img src={imgSplitView.src} alt="Bokuchi の Marp プレゼンテーション — 左にエディタの Markdown、中央にレンダリングされたスライド、右にアウトラインパネル" width="700" />

## Marp を有効にする

Marp はデフォルトで無効です。有効にするには：

1. **Cmd+,**（macOS）または **Ctrl+,**（Windows/Linux）で[設定](/ja/guides/settings/)を開く
2. **詳細設定** > **レンダリング拡張** を開く
3. **Marp（プレゼンテーション）** をオンにする

<Aside>
Marp が無効の場合、ライブラリは読み込まれないため、エディタのパフォーマンスに影響はありません。
</Aside>

<img src={imgEnableSettings.src} alt="Bokuchi の設定画面 — レンダリング拡張で Marp（プレゼンテーション）のトグルがオン" width="700" />

## プレゼンテーションの書き方

Markdown ファイルの先頭にある YAML フロントマターに `marp: true` を追加します。スライドの区切りには `---`（ハイフン3つを単独の行に記述）を使います。

```markdown
---
marp: true
---

# スライド 1

プレゼンテーションへようこそ！

---

# スライド 2

- ポイント A
- ポイント B
- ポイント C

---

# スライド 3

ありがとうございました！
```

フロントマターに `marp: true` が検出されると、プレビューが自動的にプレゼンテーションモードに切り替わります。

## 表示モード

Marp プレゼンテーションは、現在の表示モードによって動作が異なります：

### 分割ビュー（Ctrl+Shift+1）

すべてのスライドがプレビューペインに縦並びで表示されます。エディタとスライドが連動してスクロールするため、編集中のスライドを常に確認できます。

### プレビューのみ（Ctrl+Shift+3）

スライドが1枚ずつ表示され、ナビゲーションコントロールが使えます：

| 操作 | ショートカット |
|------|----------------|
| 次のスライド | 右矢印 / 下矢印 / Space |
| 前のスライド | 左矢印 / 上矢印 |
| フルスクリーン終了 | Escape |

上部にスライドカウンター（例：「スライド 2 / 10」）が表示されます。

<img src={imgPreviewOnly.src} alt="プレビューのみモード — Marp スライドが1枚表示され、ナビゲーション矢印とスライドカウンターがある" width="700" />

### フルスクリーン

プレビューヘッダーのフルスクリーンボタンをクリックすると、現在のスライドが16:9のアスペクト比で暗い背景に表示されます。**Escape** で終了します。

<img src={imgFullscreen.src} alt="Marp スライドが16:9・暗い背景でフルスクリーン表示され、スライドカウンターがある" width="700" />

### サムネイルグリッド

プレビューヘッダーのグリッドボタンをクリックすると、すべてのスライドを一覧で確認できます。サムネイルをクリックすると、そのスライドにジャンプします。

<img src={imgThumbnailGrid.src} alt="サムネイルグリッド — Marp プレゼンテーションの全スライドを一覧表示" width="700" />

## テーマ

Marp はビルトインテーマに対応しています。フロントマターでテーマを指定できます：

```markdown
---
marp: true
theme: gaia
---
```

利用可能な Marp テーマは `default`、`gaia`、`uncover` などがあります。

<img src={imgThemeGaia.src} alt="gaia テーマでレンダリングされた Marp プレゼンテーション — フロントマターの theme: gaia で指定" width="700" />

## カスタムテーマ

内蔵テーマに加えて、CSS ファイルを置いたフォルダから独自の Marp テーマを登録できます。

1. [設定](/ja/guides/settings/) > **詳細** を開きます（Marp を有効にしておく必要があります）。
2. **Marp テーマフォルダ** で **フォルダを選択…** をクリックし、`.css` テーマファイルが入ったフォルダを選びます。
3. Bokuchi がフォルダを読み取り、見つかったテーマを一覧表示します。ファイルを追加・変更したら **再読み込み**、フォルダを外すには **クリア** を使います。

各 CSS ファイルは、先頭にテーマ名を記した `@theme` ヘッダーがあれば、選択できるテーマになります。

```css
/* @theme my-theme */

section {
  background: #fdf6e3;
  color: #333;
}
```

あとは内蔵テーマと同じように、スライドのフロントマターでその名前を指定します。

```markdown
---
marp: true
theme: my-theme
---
```

<img src={imgThemeFolder.src} alt="設定 > 詳細 の「Marp テーマフォルダ」カード。フォルダを選択／再読み込み／クリアのボタンと、検出されたテーマ名の一覧が見える" width="700" />

<Aside>
`@theme` ヘッダーがないファイルは無視されます。Bokuchi はそれらを別に一覧表示するので、なぜ取り込まれなかったかが分かります。
</Aside>

## 画像

Markdown 内の相対パスの画像は Marp プレゼンテーション内でも動作します。Bokuchi が自動的にパスを解決し、プレビューで正しく画像を表示します。

```markdown
---
marp: true
---

![図表](./images/diagram.png)
```
