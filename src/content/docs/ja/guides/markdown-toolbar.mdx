Markdown ツールバーはタブバーとエディタの間に配置されています。ボタンをクリックするだけで書式を適用でき、Markdown の記法を覚える必要はありません。

<img src={imgToolbar.src} alt="Markdown ツールバーの全体表示" width="700" />

## ボタン一覧

| ボタン | 動作 | 挿入される Markdown |
|--------|------|---------------------|
| **H1 / H2 / H3** | 見出しの挿入（ドロップダウンで H1〜H6） | `# ` / `## ` / `### ` 等 |
| **B** | 太字 | `**テキスト**` |
| **I** | 斜体 | `*テキスト*` |
| **S** | 取り消し線 | `~~テキスト~~` |
| **箇条書き** | 箇条書きリスト | `- 項目` |
| **番号付き** | 番号付きリスト | `1. 項目` |
| **チェック** | タスクリスト | `- [ ] 項目` |
| **リンク** | リンクの挿入 | `[テキスト](url)` |
| **画像** | 画像の挿入 | `![代替テキスト](url)` |
| **コード** | コードブロック | ` ``` ` |
| **引用** | 引用 | `> テキスト` |
| **罫線** | 水平線 | `---` |
| **テーブル** | テーブルテンプレートの挿入 | パイプ形式の表 |
| **テーブルを整形** | カーソル位置のテーブルを整形してビジュアル編集（**Cmd/Ctrl+Shift+L**） | — |

## 使い方

1. **テキスト未選択時** — ボタンをクリックすると、カーソル位置にプレースホルダー付きの書式が挿入されます
2. **テキスト選択時** — テキストを選択してからボタンをクリックすると、選択範囲に書式が適用されます

<Aside>
Markdown 初心者の方には特にツールバーが便利です。まずテキストを書いてから、装飾したい部分を選択してボタンを押す使い方がおすすめです。
</Aside>

## テーブルを編集する

**テーブルを整形** ボタンを押すと、Bokuchi のビジュアルなテーブルエディタが開きます。パイプを手で揃えなくても、行や列の追加、配置の指定、セルの編集ができます。詳しい手順は[テーブル編集](/ja/guides/table-editing/)をご覧ください。

## ツールバーを隠す

エディタをすっきりさせたい場合は、[設定](/ja/guides/settings/) > エディタ > 整形ツールバー でツールバーを隠せます。[集中モード](/ja/guides/focus-mode/)の間も自動的に隠れます。
