フォルダツリーパネルを使うと、Bokuchi の中からディレクトリを閲覧し、ファイルを直接開くことができます。

<img src={imgFolderTree.src} alt="フォルダツリーパネルにディレクトリ構造が表示されている様子" width="700" />

## フォルダツリーを開く

**Ctrl+Shift+E** でフォルダツリーパネルの表示/非表示を切り替えます。

パネル内のボタンまたはメニューから、閲覧するディレクトリを選択します。

## 機能

- **ファイル閲覧** — フォルダ構造をツリー形式でナビゲーション
- **クリックで開く** — ファイルをクリックすると新しいタブで開きます
- **遅延読み込み** — サブディレクトリは必要な時に読み込まれるため、動作が軽快です
- **アクティブファイルのハイライト** — 現在開いているファイルがツリー内でハイライト表示されます
- **ファイルフィルター** — Markdown ファイルのみ、またはすべてのファイルを表示
- **名前の変更** — ファイルを右クリックして、コンテキストメニューからリネームできます

### ファイル名の変更

ツリー内のファイルを右クリックすると、コンテキストメニューが表示されます。**Rename** を選択してください。

<img src={imgRename01.src} alt="フォルダツリーの右クリックメニューに Rename が表示されている様子" width="700" />

新しいファイル名を入力するダイアログが表示されます。

<img src={imgRename02.src} alt="ファイル名変更ダイアログ" width="700" />

<Aside>
Markdown ファイルのみの表示とすべてのファイルの表示は、パネル内のフィルターボタン、または[設定](/ja/guides/settings/) > インターフェース > ファイルフィルター で切り替えできます。
</Aside>

## 表示モード

| モード | 説明 |
|--------|------|
| **常時表示（Persistent）** | サイドパネルとして常に表示 |
| **オーバーレイ（Overlay）** | コンテンツの上に重ねて表示 |
| **OFF** | パネルを非表示 |

[設定](/ja/guides/settings/) > インターフェース > エクスプローラー表示モード で変更できます。

## 統合サイドバー

[垂直タブ](/ja/guides/tabs/)を使用している場合、フォルダツリーとタブリストを一つのリサイズ可能なサイドバーに統合して表示できます。
