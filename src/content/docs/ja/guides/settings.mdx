**Cmd+,**（macOS）または **Ctrl+,**（Windows/Linux）で設定パネルを開きます。**F1** のヘルプメニューからもアクセスできます。

## 設定カテゴリ

### エディタ

| 設定 | 説明 | デフォルト |
|------|------|------------|
| フォントサイズ | エディタの文字サイズ | 14 |
| 行番号 | 行番号を表示 | ON |
| タブサイズ | タブのスペース数 | 2 |
| 折り返し | 長い行をエディタ幅で折り返す | ON |
| ミニマップ | ドキュメント全体の縮小表示 | ON |
| 整形ツールバー | エディタ上部に [Markdown 整形ツールバー](/ja/guides/markdown-toolbar/)を表示 | ON |

<img src={imgEditor.src} alt="設定画面 — エディタタブ" width="700" />

### 外観

| 設定 | 説明 | デフォルト |
|------|------|------------|
| テーマ | 9種類の内蔵テーマから選択 | Default |

各テーマの詳細は[テーマの変更](/ja/guides/themes/)をご覧ください。

### インターフェース

| 設定 | 説明 | デフォルト |
|------|------|------------|
| 言語 | UI言語（14言語対応） | English |
| タブレイアウト | 水平タブ / 垂直タブ | 水平 |
| タブ閉じるボタン位置 | 閉じる/ピンアイコンをタブの左側・右側に表示 | 右 |
| 新規タブボタンの位置 | 垂直タブでの新規タブ（＋）ボタンの位置: 上部 / 下部 | 上部 |
| ズームレベル | 画面のズーム | 100% |
| スクロール同期 | エディタ→プレビュー / 双方向 / オフ | 双方向 |
| アウトライン表示モード | 常時表示 / オーバーレイ / オフ | 常時表示 |
| エクスプローラー表示モード | 常時表示 / オーバーレイ / OFF | OFF |
| ファイルフィルター | Markdown のみ / すべてのファイル | Markdown のみ |

### 詳細設定

| 設定 | 説明 | デフォルト |
|------|------|------------|
| オートセーブ | 3秒後に自動保存 | ON |
| 空白文字の表示 | 不可視文字を表示 | OFF |
| テーブル変換 | ペースト時の変換モード: 自動 / 確認 / OFF | 確認 |
| テーブルのレイアウト（プレビュー） | プレビューと HTML エクスポートでのテーブル列幅の決め方: 均等幅 / 内容に合わせて折り返し / 内容に合わせて横スクロール | 内容に合わせる（折り返し） |

#### レンダリング拡張

| 設定 | 説明 | デフォルト |
|------|------|------------|
| KaTeX（数式） | `$...$`（インライン）や `$$...$$`（ブロック）で数式をレンダリング | ON |
| Mermaid（図表） | ` ```mermaid ` コードブロックで図表をレンダリング | OFF |
| Marp（プレゼンテーション） | `marp: true` フロントマターを持つ Markdown をスライドとしてレンダリング | OFF |

無効化した機能はライブラリを読み込まないため、エディタの軽量性が維持されます。

Marp を有効にすると、ここに **Marp テーマフォルダ** の項目も表示され、CSS ファイルのフォルダからスライド用のカスタムテーマを登録できます。詳しくは[カスタムテーマ](/ja/guides/marp/#カスタムテーマ)をご覧ください。

<Aside>
KaTeX・Mermaid・Marp の内容を含むファイルを、対応する設定がオフのまま開いた場合は、初回限定でお知らせが表示され、対応する設定への動線が案内されます。「数式や図、スライドが表示されない」原因が機能オフによるものだと一目で分かります。
</Aside>

### グローバル変数

[グローバル変数](/ja/guides/variables/)の追加・編集・削除・インポート・エクスポートができます。

## 設定のエクスポートとインポート

すべての設定を JSON ファイルとしてエクスポートし、他のマシンにインポートできます。設定のバックアップや共有に便利です。

- **エクスポート：** 設定パネルのエクスポートボタンをクリック
- **インポート：** インポートボタンをクリックし、保存済みの JSON ファイルを選択

## 設定のリセット

最初からやり直したい場合は、リセットボタンですべての設定をデフォルトに戻せます。

<Aside>
設定をリセットしても、グローバル変数は削除されません。変数は別途管理されています。
</Aside>
