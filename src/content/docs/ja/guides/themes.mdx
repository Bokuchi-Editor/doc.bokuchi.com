Bokuchi には 9 種類のテーマが内蔵されています。

## テーマ一覧

| テーマ      | 種類   | 説明                                                           |
| ----------- | ------ | -------------------------------------------------------------- |
| **Default** | ライト | ブルーアクセントのすっきりしたライトテーマ（Material Design）  |
| **Dark**    | ダーク | 暗い背景に明るい文字で、目にやさしいダークテーマ               |
| **Pastel**  | ライト | ミントグリーンとピンクを使った、柔らかく優しい配色             |
| **Vivid**   | ライト | オレンジとパープルを使った、鮮やかで大胆な配色                 |
| **暁**      | ライト | 朝焼けのような、ほのかなテラコッタ色を帯びた暖かいライトテーマ |
| **黄昏**    | ダーク | 夕暮れのような、暖かみのある金色を帯びたダークテーマ           |
| **絹**      | ライト | わずかに寒色を帯びた、洗練されたライトのモノトーン             |
| **墨**      | ダーク | わずかに寒色を帯びた、落ち着いたダークのモノトーン             |
| **Darcula** | ダーク | JetBrains IDE にインスパイアされた、落ち着いたダークテーマ     |

## テーマを変更する

テーマの変更方法は 2 つあります：

### 設定画面から

1. **設定** を開きます（macOS: Cmd+, / Windows/Linux: Ctrl+,）
2. **外観** セクションに移動します
3. お好みのテーマを選択します

### ステータスバーから

ウィンドウ下部のステータスバーに表示されている**テーマ名**をクリックすると、テーマを切り替えることができます。

## テーマの保持

テーマの設定は自動的に保存され、次回 Bokuchi を開いた際に復元されます。
