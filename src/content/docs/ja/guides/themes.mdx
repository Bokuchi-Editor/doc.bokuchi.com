Bokuchi には 5 種類のテーマが内蔵されています。

## テーマ一覧

| テーマ | 説明 |
|--------|------|
| **Default** | ブルーアクセントのすっきりしたライトテーマ（Material Design） |
| **Dark** | 暗い背景に明るい文字で、目にやさしいダークテーマ |
| **Pastel** | ミントグリーンとピンクを使った、柔らかく優しい配色 |
| **Vivid** | オレンジとパープルを使った、鮮やかで大胆な配色 |
| **Darcula** | JetBrains IDE にインスパイアされた、落ち着いたダークテーマ |

## テーマを変更する

テーマの変更方法は 2 つあります：

### 設定画面から

1. **設定** を開きます（macOS: Cmd+, / Windows/Linux: Ctrl+,）
2. **外観** セクションに移動します
3. お好みのテーマを選択します

### ステータスバーから

ウィンドウ下部のステータスバーに表示されている**テーマ名**をクリックすると、テーマを切り替えることができます。

## テーマの保持

テーマの設定は自動的に保存され、次回 Bokuchi を開いた際に復元されます。
