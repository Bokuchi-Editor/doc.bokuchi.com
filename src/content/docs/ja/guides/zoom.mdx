Bokuchi の画面全体のサイズを拡大・縮小できます。

## キーボードショートカット

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| ズームイン | Cmd+Plus (=) | Ctrl+Plus (=) |
| ズームアウト | Cmd+Minus | Ctrl+Minus |
| ズームリセット | Cmd+0 | Ctrl+0 |

## ステータスバー

ウィンドウ下部のステータスバーに現在のズームレベルが表示されています。ステータスバーのズームボタンからも調整できます。

## ズーム範囲

- **最小：** 50%
- **最大：** 300%
- **ステップ：** 10% 刻み
- **デフォルト：** 100%

## 設定の保持

ズームレベルは自動的に保存され、次回 Bokuchi を開いた際に復元されます。
